สำรองข้อมูลนักเรียน (Export Data)

คุณสามารถสำรองข้อมูลการเยี่ยมบ้านทั้งหมดลงในไฟล์ JSON เพื่อเก็บไว้สำรอง หรือนำไปกู้คืนในเครื่องอื่นได้

นำเข้าข้อมูล / กู้คืนข้อมูล (Import Data / Restore)

อัปโหลดไฟล์ `.json` เพื่อกู้คืนข้อมูลการเยี่ยมบ้าน (ข้อมูลจะถูกบันทึกทับในกรณีที่มีรหัสนักเรียนซ้ำกัน)
